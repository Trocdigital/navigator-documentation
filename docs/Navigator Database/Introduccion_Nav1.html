1. Resumen​

En esta primera sesión se vio de forma general cómo podemos partir de un query SQL a los datos a una representación visual. En las siguientes clases veremos cómo utilizar múltiples tipos de representaciones visuales y vamos a ir repasando las opciones que tienen estas representaciones visuales para cambiar todos sus parámetros internos es forma de tal de presentaciones más personalizadas.

2. Temas desarrollados

Introducción a Navigator El primer tema tratado en la transcripción sobre Introducción a navigator es la estructura de un dashboard. Un dashboard es una colección de elementos de representación visual que se distribuyen de forma organizada en una grilla. Los elementos de representación visual pueden ser gráficos, tarjetas, tablas, etc.

Cada elemento de representación visual tiene un identificador único y se puede editar desde la interfaz gráfica. Los cambios que se hagan en la interfaz gráfica se guardan en un archivo JSON que se puede modificar directamente.

Componentes de navigator​

Los componentes de navigator son los elementos que componen un dashboard y pueden ser de diferentes tipos como tarjetas, gráficos, tablas, etc.
Cada componente tiene sus propias características y se pueden personalizar de diferentes maneras.
Primero para crear un dashboard, se debe crear un query SQL que devuelva los datos que se quieren mostrar. Luego, se debe crear un widget y asignarle el query SQL.
Finalmente, se pueden personalizar los diferentes elementos del widget, como el título, los colores, los iconos, etc.
En las siguientes clases de capacitación, veremos cómo utilizar diferentes tipos de componentes y cómo personalizarlos. También veremos algunos ejemplos de dashboards que hemos creado para nuestros clientes.

Elementos visuales de navigator​

Los elementos visuales de navigator son los widgets y las tarjetas. Los widgets son los elementos que representan los datos y las tarjetas son los contenedores de los widgets.
Los widgets pueden tener diferentes tipos de representaciones visuales, como gráficos de barras, tablas de tortas, gráficos estaqueados, etc.
Las tarjetas pueden tener uno o dos elementos y se pueden usar para mostrar información adicional relacionada con los datos.
Los widgets y las tarjetas se pueden personalizar cambiando su color, icono, título, etc.

También se pueden aplicar funciones de formato a los valores numéricos para mostrarlos de diferentes maneras.

Crear dashboards​

Para crear un dashboard en navigator, primero se debe crear un query SQL que recupere los datos que se quieren mostrar.
Luego, se debe crear un widget y asignarle el query SQL.
Finalmente, se debe personalizar el widget para cambiar su apariencia, como el color, el icono, etc. Finalmente, podemos guardar los cambios del widget y visualizarlo en el dashboard.

Convertir dashboards en módulos​

Debemos ir a la sección de administración y autoayuda service y buscar el dashboard que queremos convertir.
Una vez que lo encontremos, debemos hacer clic en el botón "Convertir a módulo".
Esto creará una nueva entrada en el menú de navegación con el nombre del dashboard.

Asignarse un dashboard​

Para asignarse un dashboard, debemos ir a la sección de "Dashboards" y hacer clic en el botón "Asignar". Luego, seleccionamos el dashboard que queremos asignarnos y hacemos clic en el botón "Guardar".
Una vez que nos hemos asignado un dashboard, este se mostrará en el menú de dashboards y podremos acceder a él para visualizarlo y editarlo.

Widgets de navigator​

Los widgets de navigator son elementos visuales que permiten mostrar información de forma gráfica y dinámica.
Los widgets se pueden crear a partir de un query SQL y se pueden editar en la interfaz gráfica.
Los widgets tienen parámetros que se pueden modificar para cambiar su apariencia, como el color, el icono o el formato del valor numérico.
También se pueden añadir funciones de formato a los valores numéricos para mostrarlos de forma personalizada.
Los widgets se pueden utilizar para crear dashboards personalizados con diferentes tipos de representaciones visuales.

Editar widgets​

Para editar los widgets, primero debemos encontrar el identificador del widget. Podemos hacerlo haciendo clic en el título del widget y luego copiando el código que aparece en la parte superior de la ventana.
Una vez que tenemos el identificador del widget, podemos abrir el archivo de configuración del widget en el editor de código.
El archivo de configuración del widget contiene todos los datos sobre el widget, incluyendo su título, descripción, colores, iconos y parámetros.
Podemos editar los datos del widget cambiando los valores en el archivo de configuración.
Una vez que hemos terminado de editar los datos del widget, debemos guardar el archivo de configuración y luego actualizar el widget en el navegador.
El widget se actualizará con los nuevos datos y se mostrará en el navegador.

Cambiar el color de los widgets​

Para cambiar el color de los widgets podemos hacerlo desde la interfaz gráfica, editando el widget y modificando el parámetro class. Dentro del parámetro class podemos colocar un código hexadecimal de color, o utilizar alguna de las clases de colores permitidas por el sistema.

Cambiar el icono de los widgets​

Para cambiar el icono de los widgets, debemos ir a la sección de widgets, buscar el widget que queremos editar, y en el código JSON, cambiar el valor del atributo "class" por el nombre del icono que queremos utilizar. Por ejemplo, si queremos utilizar el icono de diamante, debemos poner el valor "fa-diamond".

Aplicar funciones de formato a los valores numéricos​

Para aplicar funciones de formato a los valores numéricos podemos utilizar la función format number o format percent. La función format number nos permite multiplicar el valor numérico por 100 y añadir el símbolo de porcentaje. La función format percent nos permite añadir el símbolo de porcentaje al valor numérico sin multiplicarlo por 100.

3. Preguntas más frecuentes

¿Qué es un dashboard?
 Un dashboard es una colección de elementos de representación visual que nos permite visualizar información de forma gráfica y tabular.

¿Cómo puedo crear un dashboard?
 Para crear un dashboard, primero debes crear un query SQL que contenga los datos que deseas mostrar. Luego, puedes utilizar la interfaz gráfica de navigator para crear un dashboard y agregar los widgets que desees.

¿Cómo puedo editar un dashboard?
 Para editar un dashboard, puedes utilizar la interfaz gráfica de navigator. Puedes cambiar el nombre del dashboard, el título del dashboard, la descripción del dashboard, y los parámetros del dashboard.

Para editar un dashboard, debes seguir estos pasos:

  1. Haz clic en el botón "Editar" en la parte superior del dashboard.
  2. Haz los cambios que desees en el dashboard.
  3. Haz clic en el botón "Guardar" para guardar los cambios.

¿Cómo puedo convertir un dashboard en un módulo?
 Para convertir un dashboard en un módulo, debes seguir estos pasos:

  1. Haz clic en el botón "Convertir en módulo" en la parte superior del dashboard.
  2. En la ventana emergente, ingresa un nombre para el módulo y haz clic en "Guardar".
  3. El dashboard se convertirá en un módulo y se moverá a la sección de módulos de la barra lateral izquierda.

¿Cómo puedo eliminar un dashboard?
 Para eliminar un dashboard, debes seguir estos pasos:

  1. Haz clic en el botón "Eliminar" en la parte superior del dashboard.
  2. En la ventana emergente, haz clic en "Sí" para confirmar que deseas eliminar el dashboard.

¿Cómo puedo compartir un dashboard con otros usuarios?
 Para compartir un dashboard en un módulo, primero debes crear el dashboard y luego convertirlo en un módulo.
Para convertir un dashboard en un módulo, ve a la sección de módulos y haz clic en el botón "Nuevo módulo".
En el campo "Nombre del módulo", ingresa el nombre del módulo que deseas crear. En el campo "Descripción del módulo", ingresa una descripción del módulo.
En el campo "Dashboard", selecciona el dashboard que deseas convertir en un módulo. Haz clic en el botón "Guardar" para crear el módulo.

Una vez que hayas creado el módulo, puedes compartirlo con otros usuarios haciendo clic en el botón "Compartir".
En el campo "Usuarios", ingresa los nombres de los usuarios con los que deseas compartir el módulo.
En el campo "Permisos", selecciona los permisos que deseas otorgar a los usuarios. Haz clic en el botón "Guardar" para compartir el módulo.

¿Cómo puedo ver los dashboards compartidos conmigo?
 Para ver los dashboards compartidos contigo, ve a la sección de módulos y haz clic en el botón "Compartido conmigo". En esta sección, se mostrarán todos los módulos que han sido compartidos contigo. Haz clic en el nombre de un módulo para abrirlo.

¿Cómo puedo cambiar el formato de un valor numérico en un dashboard?
 Para cambiar el formato de un valor numérico en un dashboard, debes seguir estos pasos:

  1. Haz clic en el valor numérico que deseas cambiar.
  2. En el menú desplegable, selecciona el formato que deseas utilizar.
  3. Haz clic en "Aceptar" para guardar los cambios.

¿Qué es un widget?
 Un widget es un elemento de representación visual que se puede añadir a un dashboard.

¿Cómo puedo crear un widget?
 Para crear un widget, primero debes crear un query SQL que devuelva los datos que quieres mostrar en el widget. Luego, debes añadir el widget a la interfaz gráfica de Navigator y establecer el data source del widget con el query SQL que creaste.

¿Cómo puedo editar un widget?
 Para editar un widget, puedes cambiar el título, la descripción, los parámetros y el código JSON del widget.

¿Cómo puedo cambiar el color de un widget?
 Para cambiar el color de un widget, puedes utilizar la función format color.

¿Cómo puedo añadir un icono a un widget?
 Para añadir un icono a un widget, puedes utilizar la función format icon.

¿Cómo puedo añadir una función de formato a un widget?
 Para añadir una función de formato a un widget, puedes utilizar la función format function.

¿Puedo utilizar código hexadecimal para el color de un widget?
 No, se debe utilizar una clase de color permitida por el sistema.

¿Cómo puedo cambiar el icono de un widget?
 Se puede cambiar el icono por uno de los iconos de la galería de Font Awesome.

¿Cómo puedo añadir un mensaje de alerta a un dashboard?
 Se puede añadir un mensaje de alerta utilizando el widget de mensaje.

¿Cómo puedo agregar un widget a un dashboard?
 Para agregar un widget a un dashboard, puedes utilizar la interfaz gráfica de navigator. Puedes arrastrar y soltar el widget en el dashboard.

¿Cómo puedo eliminar un widget de un dashboard?
 Para eliminar un widget de un dashboard, puedes utilizar la interfaz gráfica de navigator. Haz clic en el widget y luego haz clic en el botón "Eliminar".
